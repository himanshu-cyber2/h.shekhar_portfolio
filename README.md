H.SHEKHAR | Digital Architect Portfolio

A high-performance, visually immersive portfolio website designed with an "Ice-Carbon" aesthetic. This project showcases a blend of creative frontend engineering, 3D simulations, and interactive logic modules.

üöÄ Live Modules

1. Quantum Ledger (Expense Tracker)

A financial intelligence module that allows users to track expenditures.

Features: Real-time data visualization using Chart.js, persistent session logging, and category-based breakdown.

Logic: Implements a functional data-store and dynamic doughnut chart updates.

2. AIRA Typo Engine

A neural input simulation (typing game) designed to test user speed and accuracy.

Features: 30-second countdown, real-time WPM (Words Per Minute) calculation, and accuracy tracking.

Interactions: Custom keyboard event listeners with visual feedback for correct/incorrect inputs.

3. Logical Conflict Resolver (RPS)

An interactive implementation of Rock Paper Scissors.

Features: Score tracking and randomized computer logic.

Design: Minimalist icon-based UI with glowing state transitions.
